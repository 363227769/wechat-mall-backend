## 微信小程序商城-服务端

本项目是 [wechat-mall](https://github.com/ZuoFuhong/wechat-mall-miniapp) 微信小程序商城 配套的`服务端`

## Deployment

```sh
# clone the project
git clone git@github.com:ZuoFuhong/wechat-mall-backend.git

# update dependency
go mod tidy

# build the project
./build.sh
```
